GreenTech Solutions is a responsive and accessible multi-page website focused on residential wind turbine technology and sustainable energy solutions. The website was created as a final project for Web Development and highlights renewable energy products, educational resources, and interactive features designed to engage users and promote environmental sustainability.

This project was developed using HTML5, CSS3, and JavaScript. Additional tools and technologies include GitHub for version control, GitHub Pages for deployment, and Visual Studio Code for development and testing.

To run the project locally, download or clone the repository from GitHub and open the project folder in Visual Studio Code. Open the `index.html` file in a web browser or use the VS Code Live Server extension for a local development server. The website is fully responsive and can be viewed on desktop, tablet, and mobile devices.
