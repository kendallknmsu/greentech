Final Project Reflection

This website was developed as a final project. The purpose of the project was to create a responsive and accessible website for GreenTech Solutions focused on residential renewable energy systems and sustainability education.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • GitHub
  • GitHub Pages
  • Visual Studio Code

Reflection

This project helped strengthen my understanding of responsive web design, accessibility standards, and JavaScript functionality. One of the biggest challenges was ensuring the website displayed properly across multiple screen sizes and devices. Using CSS Grid, Flexbox, and media queries improved the overall user experience while keeping the design modern and organized.
